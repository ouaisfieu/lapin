TL;DR — comment jouer en 30 secondes
  • 1. Ouvre Zalagtiga pour l’ambiance et le briefing.
  • 2. Ouvre le terminal Minitel & MAP.
  • 3. Sur le Minitel, entre le code ↑ ↑ ↓ ↓ ← → ← → B A pour débloquer la MAP.
  • 4. Clique sur des cases de l’échiquier, remplis les fiches (acteurs, lieux, médias, hypothèses…)
  • 5. Exporte ta carte (JSON / CSV / Markdown), partage-la ou réimporte-la pour rejouer.
  • 6. En équipe, répartissez-vous les factions et les zones de l’échiquier.

Tout se passe en local dans ton navigateur. Rien n’est envoyé sur un serveur : c’est un jeu pédagogique, pas un outil de surveillance.

Étapes logiques · de A à Z

Voici le chemin “officiel” pour jouer, que tu sois en solo ou en équipe. Les liens supposent que tu as nommé les fichiers comme suit :

  • zalagtiga.html — borne d’arcade / briefing
  • minitel-boot-map.html — Minitel, Konami, MAP échiquier
  • monitor.html — hub / faux minitel / annuaire (optionnel)
  • recap-arcade.html — récap du jeu (optionnel)

1. Lancer la partie (ambiance & briefing)

  1. Ouvre Zalagtiga dans ton navigateur.
  2. Lis l’intro sur l’écran CRT : Place Poelaert, années 80, guerre froide, salle de jeux glauque…
  3. Cliques sur INSERT COIN pour afficher le briefing de mission.

Objectif : comprendre que tu vas jouer avec tes habitudes numériques, pas avec des “dossiers” sur des gens.

mise en contexte fiction hygiène numérique

2. Ouvrir le terminal Minitel et débloquer la MAP

  1. Ouvre Minitel & MAP.
  2. Laisse défiler le faux écran d’allumage (boot).
  3. Quand le boot est terminé, entre le Konami code sur ton clavier : ↑ ↑ ↓ ↓ ← → ← → B A.
  4. La MAP d’enquête (échiquier 8×8) apparaît.

Tu viens de passer de “spectateur” à “joueur/joueuse” : le code secret marque ton consentement à jouer.

3. Comprendre l’échiquier d’enquête

L’échiquier 8×8 représente une carte mentale des enjeux qui t’intéressent. Par exemple :

  • ligne 8 : acteurs nationaux / internationaux
  • ligne 4–5 : niveau local / régional
  • colonne A–D : institutions publiques / associations
  • colonne E–H : médias / entreprises / “zones floues”

Tu peux décider seul·e de ce que signifie chaque rangée / colonne, ou le faire en équipe.

4. Remplir les fiches (cases de l’échiquier)

En cliquant sur une case (ex : C5), une fiche s’ouvre :

  • Nom / acteur principal — ex : “Association Quartier X”, “Media Y”.
  • Type — acteur, organisation, média, lieu, événement, hypothèse…
  • Risque — faible / moyen / élevé (niveau de sensibilité du sujet).
  • Tags — mots-clés : “logement”, “climat”, “algorithmes”, “emploi”…
  • Lien principal — site officiel, article de référence, ressource clé.
  • GPS — coordonnées d’un lieu public (pas d’adresses privées).
  • Image (URL) — logo, photo libre, carte, schéma…
  • Notes — contexte, questions, signaux faibles, idées de veille.

Après enregistrement, la case s’allume avec un point “●” et peut afficher son image en fond. Au survol, un résumé apparaît (tooltip).

5. Sauvegarder, partager, rejouer

Les boutons JSON / CSV / Markdown te permettent d’exporter ton échiquier. Tu peux ensuite :

  • ouvrir le CSV dans un tableur (LibreOffice, Excel) pour trier, filtrer, colorer ;
  • utiliser le JSON dans un script ou un autre outil ;
  • coller le Markdown dans ton système de notes (Obsidian, Hugo…).

Tu peux aussi importer un JSON (fichier ou contenu de la zone de texte) pour reprendre une partie ou partager une configuration entre plusieurs personnes.

Mode Solo · apprendre à voir le jeu

Le mode solo sert surtout à te faire la main, sans pression. Tu peux le jouer comme un journal de bord interactif.

Étape Ce que tu fais Ce que tu y gagnes
1. Lancer Tu lis le briefing sur Zalagtiga. Tu te mets mentalement en mode “jeu” avant d’attaquer le sérieux.
2. Cartographier Tu ouvres la MAP et remplis quelques cases avec des sujets qui te concernent. Tu clarifies ce qui te pèse / t’intéresse dans l’actualité et ta vie numérique.
3. Relire Tu relis tes fiches après quelques jours, tu ajoutes des notes, tu ajustes les risques. Tu vois ce qui a vraiment de l’importance pour toi et ce qui t’angoisse “pour rien”.
4. Exporter Tu exportes ta carte en Markdown ou CSV et tu la classes dans tes dossiers. Tu gardes une trace de ton parcours, sans dépendre d’un service externe.

Mode Équipe · coopératif et modulable

En équipe, le jeu devient un atelier : idéal pour un groupe d’ami·es, une asso, une classe, un collectif de quartier, etc.

Organisation simple

  • 1 personne joue le rôle de “gardien·ne de la machine” (maître du jeu) :
    • affiche Zalagtiga sur grand écran ;
    • manipule la MAP sur un ordi projeté ;
    • exporte / importe les sauvegardes.
  • Le reste du groupe se répartit les cases / zones / factions (voir ci-dessous).

Principe de base

  1. Choisir un thème commun (ex : logement, numérique, climat, emploi…).
  2. Attribuer 2–4 cases de l’échiquier à chaque personne ou petit groupe.
  3. Remplir les fiches ensemble, en expliquant à voix haute ce qu’on ajoute et pourquoi.
  4. Discuter des liens possibles entre les cases (qui influence quoi, à quel niveau, comment ?).

Tu peux ensuite utiliser recap-arcade.html comme pense-bête pour vérifier que tout le monde a suivi les grandes étapes.

Factions & rôles possibles

Les factions servent à mettre du jeu de rôle dans l’exercice et à répartir les tâches. Tu peux les renommer / adapter à ton public.

1. Veilleurs de Poelaert
Rôle : surveiller le “paysage” global.

Ils/elles s’occupent surtout des cases du haut de l’échiquier (ligne 8–7–6) : institutions nationales, acteurs internationaux, grandes entreprises, grands médias.

Idéal pour les gens qui aiment prendre de la hauteur, lire la presse, regarder les tendances.
2. Cartographes de Quartier
Rôle : ancrer la carte dans le réel local.

Ils/elles se concentrent sur les lignes 3–4–5 : communes, associations, lieux de vie, conflits locaux, initiatives citoyennes, etc.

Idéal pour celles et ceux qui connaissent bien le terrain, les maisons de quartier, les cafés, les squats…
3. Archivistes du Minitel
Rôle : garder une mémoire claire de ce qui est saisi.

Ils/elles vérifient que chaque fiche est compréhensible, sourcée, non diffamatoire, et surveillent les exports (JSON, CSV, Markdown).

Idéal pour les gens rigoureux, qui aiment ranger, documenter, faire des synthèses propres.
4. Script-kiddies Repentis
Rôle : poser les questions gênantes.

Ils/elles se chargent d’imaginer les “dérives possibles” du système: que se passerait-il si quelqu’un utilisait ces outils pour de mauvaises raisons ?

Idéal pour les joueurs/joueuses qui aiment les scénarios, la SF, les questions de vie privée et de sécurité.

Tu n’es pas obligé d’utiliser les factions, mais elles peuvent aider un groupe à comprendre que nobody plays God : chaque rôle a ses limites.

Arbre de décision · comment je joue concrètement ?

Cet arbre simplifié t’aide à choisir ton mode de jeu et tes priorités. Clique sur les blocs pour dérouler les options.

Je joue seul·e, et je suis plutôt débutant·e solo · débutant
  • 1️⃣ Ouvre Zalagtiga une fois, lis le briefing, puis ferme la page.
  • 2️⃣ Ouvre Minitel & MAP, lance le Konami code.
  • 3️⃣ Remplis 4 à 6 cases maximum sur des sujets que tu comprends déjà bien.
  • 4️⃣ Exporte en Markdown, colle dans un fichier journal.md.
  • 5️⃣ Revient plus tard, efface / modifie ce qui ne te paraît plus pertinent.
Je joue seul·e, mais je veux un niveau “hardcore” solo · avancé
  • 1️⃣ Décide d’un thème précis (ex : “plateformes vidéo”, “logement étudiant”, “énergie”).
  • 2️⃣ Attribue toi-même une signification à chaque ligne et chaque colonne.
  • 3️⃣ Remplis au moins 16 cases, en prenant le temps de vérifier tes sources.
  • 4️⃣ Export JSON + CSV, expérimente avec un tableur et, si tu sais, un petit script.
  • 5️⃣ Écris une page de synthèse dans un autre fichier (site statique, blog, etc.) en citant le jeu comme “maquette”.
On joue en équipe, sans pression équipe · découverte
  • 1️⃣ Une personne projette Zalagtiga + Minitel & MAP sur un écran.
  • 2️⃣ On choisit un thème et on se répartit 2–3 cases chacun.
  • 3️⃣ Chacun explique à voix haute ce qu’il/elle inscrit dans sa fiche.
  • 4️⃣ On exporte en CSV, on le relit ensemble pour voir les patterns qui apparaissent.
  • 5️⃣ On termine par un tour de table : “qu’est-ce que j’ai appris / qu’est-ce que ça change pour moi ?”.
On joue en équipe, pour produire un vrai livrable équipe · avancé
  • 1️⃣ On définit à l’avance un livrable (article, mini-dossier, page web pédagogique…).
  • 2️⃣ On utilise les factions pour répartir les responsabilités (Veilleurs, Cartographes, Archivistes, Script-kiddies).
  • 3️⃣ On joue plusieurs “tours” (sessions), en sauvegardant entre chaque avec l’export JSON.
  • 4️⃣ Après 2–3 sessions, on utilise les exports pour construire le livrable (tableaux, graphiques, récit, etc.).
  • 5️⃣ On inclut une section explicite : “ce que ce jeu n’est PAS” (ni flicage, ni vérité absolue).

FAQ · questions qu’on va te poser

Est-ce que ce jeu collecte des données sur les gens ?

Non. Tout tourne dans ton navigateur, en local. Les fiches sont stockées en localStorage et les exports sont des fichiers que toi décides d’enregistrer ou pas.

Est-ce qu’on peut mettre des personnes privées dans les cases ?

Non. On reste sur des acteurs publics, des organisations, des idées, des lieux publics, des médias, des scénarios. Le jeu est une maquette d’éducation populaire, pas un ficher.

À quoi ça sert, concrètement ?

  • À ralentir la manière dont on consomme les infos.
  • À cartographier ce qui compte pour toi / ton groupe.
  • À apprendre à documenter, sourcer, vérifier, relier.
  • À tester des méthodes de veille citoyenne, sans enjeu policier.

Est-ce que je peux modifier les règles, les factions, les pages ?

Oui, c’est même encouragé. Tu peux :

  • dupliquer ce fichier de guide et adapter le texte à ton public ;
  • changer les noms de factions, ajouter des rôles ;
  • créer d’autres cartes (un deuxième échiquier, un réseau en grille, etc.).

Tu peux considérer ce guide comme un “manuel de base” : libre à toi d’en faire un manuel maison, estampillé avec ton projet.
